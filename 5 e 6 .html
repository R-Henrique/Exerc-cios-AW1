K02 - Desenvolvimento Web com HTML, CSS e Javascript

Atualmente, praticamente todos os sistemas corporativos possuem interfaces web. Para que deseja atuar no mercado
de desenvolvimento de software, é obrigatório o conhecimento das linguangens HTML, CSS e Javascript

Esssas linguagens são utilizadas para o desenvolvimento da camada de apresentação das aplicações web.

Pré-requesitos

  • familiaridade com algum sistema operacional (Windowns/Linux/Mac OS X)
  • lógica de programação
